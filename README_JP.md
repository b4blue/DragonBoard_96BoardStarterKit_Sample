## センサーの接続
このサンプルでは、以下の組み合わせでセンサーをつなぐ想定です。

- タッチセンサー -> D1
- LED -> D2
- ボタン -> D3
- 傾きセンサー -> D4
- 温度センサー -> ADC1
- 照度センサー -> ADC2

## 利用しているセンサー 
#### 出力
- LED: 赤 LED および二つのオンボード緑 LED

#### 入力: スイッチ
- タッチセンサー: センサーにタッチすると、各種 LED が光ります。
- ボタン: ボタンを押すと、各種 LED が光ります。
- 傾きセンサー: センサーを一方向に傾けると、各種 LED が光ります。

#### 入力: センサー
- 温度センサー: 温度を測定
- 照度センサー: 照度を測定
